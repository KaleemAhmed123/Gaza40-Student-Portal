# Gaza40Plus Architecture Skill

This Anthropic Skill is designed to assist agents and developers in understanding, navigating, and maintaining the architecture of the Gaza40Plus platform.

## Structure
* `SKILL.md`: The core instructions, triggering rules, and layout references for the Gaza40Plus platform.
* `architecture/project-structure.md`: Detailed guide of the frontend codebase directories, feature modules, and reusable folders.
* `architecture/backend.md`: Detailed guide of the backend modules, shared directories, middlewares, routes, and services.
* `architecture/frontend.md`: Summary of frontend routing, layout, and component architecture patterns.
* `architecture/development-principles.md`: List of key engineering and development principles for code changes.
* `references/`: (Optional placeholder) Directory for detailed architectural sub-documents.
* `scripts/`: (Optional placeholder) Directory for architectural linters, scanners, or analysis utilities.
* `assets/`: (Optional placeholder) Visual diagrams or architectural mockups.

## Usage
This skill triggers automatically when questions about the Gaza40Plus architecture, directory structure, components, or patterns are asked.
