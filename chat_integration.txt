Based on the call you had with hamza, below is our current understanding of the Chat feature requirements. 
Please review and confirm whether this aligns with your expectations? If any changes or corrections are needed, kindly update them here.

Real-Time Chat Feature

A real-time chat system will be introduced to facilitate communication between Admins, Regional Admins, and Mentors with role-based access control.

Direct Chat

Admin

- Admin can initiate one-to-one chats with any Regional Admin or Mentor.
- Admin can search and communicate with all users across all regions.
- User search should support filters such as University and Country.

Regional Admin

- Regional Admins can search and directly chat with:
  - Mentors belonging to their own region.
  - Other Regional Admins, including those from different regions.
  - Admin.
- The system should support multiple Regional Admins within the same region to ensure future scalability.

Mentor

- Mentors can search and directly chat with:
  - Their Regional Admin(s). If multiple Regional Admins exist within the same region, mentors can select the desired Regional Admin.
  - Admin.
- Mentors cannot chat directly with other Mentors.

Group Chat

Regional Admin

- Regional Admins can create groups.
- They can add:
  - Mentors belonging to their own region only.
  - Other Regional Admins, including those from different regions.
- Regional Admins cannot add mentors from other regions.

Admin

- Admin can create groups without regional restrictions.
- Admin can add any Regional Admin or Mentor to a group.
- Admin can search any user or group and is, by default, a member of all groups on the platform.

Group Permissions

Group management will follow WhatsApp-style permissions:

- The creator of a group becomes the Group Admin.
- Group Admins can:
  - Add participants.
  - Remove participants.
  - Delete messages.
  - Manage group settings and permissions.

Restrictions

- Mentors cannot initiate or participate in one-to-one chats with other Mentors.
- Mentors cannot create groups.
- Regional Admins can only create groups containing mentors from their own region.
- Admin has unrestricted access to direct chats and group creation.

Please let us know if anything needs to be added, removed, or modified.
